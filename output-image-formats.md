# Форматы выходных изображений

### TIFF (16-битный)

16-битный (цифровое значение пикселя 0–65535) выходной формат TIFF (.tif).

### TIFF (32-разрядный, проценты %)

32-разрядный (плавающая точка пикселя 0,0–1,0) TIFF (.tif) формат вывода. Значения пикселей представляют проценты отражения, где 0,0 = 0 % и 1,0 = 100 %.

### **JPG (8-битный)**

8-битный (цифровое значение пикселя 0–255) выход в формате JPG (.jpg).

### **PNG (8-битный)**

Вывод в формате PNG (.png) с 8-битным (цифровое значение пикселя 0–255) разрешением. Сжатие без потерь, подходящее для просмотра в Интернете и визуализации.
