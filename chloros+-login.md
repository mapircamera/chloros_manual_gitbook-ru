## Вход в интерфейс командной строки

Войдите в систему, используя свои учетные данные Chloros+, чтобы включить обработку CLI.

**Синтаксис:**

```bash
chloros-cli login <email> <password>
```

**Пример:**

```powershell
chloros-cli login user@example.com 'MyP@ssw0rd123'
```

{% подсказка style="предупреждение" %}
**Специальные символы**. Используйте одинарные кавычки для паролей, содержащих такие символы, как `$`, `!` или пробелы.

### Срок действия плана

Срок действия плана в графическом интерфейсе показывает, когда ваша лицензия станет недействительной. Срок действия повторяющихся ежемесячных подписок истекает в конце месяца. Для годовых подписок — это год после начала подписки. Для проверки лицензии требуется ежемесячное подключение к Интернету с 30-дневным льготным периодом.

### Ограничение количества устройств

Каждый план Chloros+ предлагает разное количество зарегистрированных устройств. Каждое устройство, на которое вы входите с помощью учетной записи Chloros+, будет учитываться в количестве зарегистрированных устройств. Вы можете переименовать и удалить устройство на странице своей учетной записи MAPIR Cloud.
