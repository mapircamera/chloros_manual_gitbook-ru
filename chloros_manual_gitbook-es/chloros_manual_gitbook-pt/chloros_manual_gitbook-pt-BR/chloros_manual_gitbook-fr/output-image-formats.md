# Форматы выходного изображения

### TIFF (16-битный)

16-битный (цифровое число пикселей 0–65535) выходной формат TIFF (.tif).

### TIFF (32 бит, в процентах %)

Вывод 32-битного (пиксели с плавающей запятой 0,0–1,0) формата TIFF (.tif). Значения пикселей представляют процент отражения, где 0,0 = 0% и 1,0 = 100%.

### **JPG (8 бит)**

8-битный (цифровое число пикселей 0–255) выходной формат JPG (.jpg).

### **PNG (8-битный)**

8-битный (цифровое число пикселей 0–255) выходной формат PNG (.png). Сжатие без потерь, подходящее для просмотра и визуализации в Интернете.
