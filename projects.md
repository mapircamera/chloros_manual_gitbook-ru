# Графический интерфейс пользователя: Проекты

Chloros позволяет создавать проекты, которые можно будет открыть в будущем.

## Новый проект

<figure><img src=".gitbook/assets/project_new.JPG" alt=""><figcaption></figcaption></figure>Выберите «Новый проект» в главном меню и введите уникальное имя для вашего проекта.

## Открыть проект

<figure><img src=".gitbook/assets/project_open.JPG" alt=""><figcaption></figcaption></figure>Выберите «Открыть проект», чтобы увидеть список существующих проектов в папке «Проекты». Если проектов нет, вторичное боковое меню не откроется. На фотографии выше вы можете увидеть несколько проектов, созданных с помощью GUI (t1, t2, t3). Проекты DATE\_TIME были созданы с помощью CLI с использованием стандартной схемы именования проектов. Щелкнув по любому имени проекта, вы откроете его.

Нажатие кнопки «Открыть папку проекта» откроет проводник файлов вашего компьютера по пути к проекту. Вы можете настроить путь к проекту в [Настройках проекта](project-settings/project-settings.md).
